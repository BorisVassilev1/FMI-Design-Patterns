# Design Patterns Homework 1

## Compilation instructions
```bash
./configure.sh
cmake --build ./build -j8 -t tests main test
```
The compiled tests are `./build/tests` and the main executable is `./main`.

To produce the test coverage report, run
```bash
./cov.sh
```
